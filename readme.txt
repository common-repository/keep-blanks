== Description ==

This plugin helps you to keep the blanks in visual editor (TinyMCE).
If you start paragraphs with spaces, you will find this plugin helpful.
The code of this plugin is modified from stringfold's TinyMCE Entities Patch, which is no longer updated.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
